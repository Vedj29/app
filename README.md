# AI Career Guidance and Skill Gap Platform

An interactive web application built using Streamlit that helps users explore technology career paths, analyze skill gaps, understand salary growth trends, and view industry demand insights.

This platform is designed for students and beginners who want structured career guidance in fields like AI, Data Science, Web Development, Cyber Security, Java, and C++.

---

## Features

### Skill Gap Analysis
- Enter your current skills
- Compare them with required skills for selected career
- Identify missing skills
- View skill match percentage with interactive pie chart

### Salary Insights
- Visualize 5-year salary growth
- Interactive line chart
- Experience vs salary comparison (LPA)

### Industry Trends
- View demand growth trends for selected career
- Area chart visualization
- Market demand progression

### Career Chatbot
- Predefined FAQ-based chatbot
- Structured and detailed answers
- No random AI responses
- Clean and controlled interaction

## Installation Guide (Run Locally)

1. Clone the repository:

git clone https://github.com/your-username/career-guidance.git

2. Navigate into the project folder:

cd career-guidance

3. Install required dependencies:

pip install -r requirements.txt

4. Run the application:

streamlit run app.py

The application will open in your browser at:
http://localhost:8501
